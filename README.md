# vscode-lookml-extension

## installation

1. download VISX file

    ```bash
    curl -sSO https://github.com/nkomiya/test/releases/download/v0.1.0/vscode-lookml-extension-0.0.1.vsix
    ```

2. install extension

    ```bash
    code --install-extension vscode-lookml-extension-0.0.1.vsix
    ```
